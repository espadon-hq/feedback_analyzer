# Система аналізу відгуків користувачів

Веб-застосунок для збору та автоматичного аналізу тональності англомовних відгуків.
Визначає чи відгук позитивний, негативний або нейтральний і зберігає результати в базі даних.

---

## Технології

| Компонент | Технологія |
|---|---|
| Веб-фреймворк | FastAPI |
| Аналіз тональності | VADER (vaderSentiment) |
| База даних | SQLite + SQLAlchemy |
| Шаблони | Jinja2 + Tailwind CSS |
| Читання CSV | pandas |
| Тести | pytest + tox |

## Як користуватись

### Додати відгук вручну
1. Відкрити головну сторінку `/`
2. Ввести текст відгуку англійською у форму
3. Натиснути **Аналізувати та зберегти**
4. Результат з'явиться у таблиці нижче

### Імпортувати відгуки з CSV
1. Перейти на сторінку `/import`
2. Завантажити CSV файл (максимум 5 MB, кодування UTF-8)
3. Обрати колонку що містить текст відгуку
4. Натиснути **Імпортувати відгуки**

Підтримувані формати CSV:

| Формат | Колонка з текстом |
|---|---|
| Власний | `text` |
| Amazon | `review_body` |
| Trustpilot | `Review Text` |

### Переглянути статистику
Перейти на сторінку `/stats` — там відображається кількість та відсоток
позитивних, негативних і нейтральних відгуків.

### Фільтрація відгуків
На головній сторінці обрати тональність у випадаючому списку
та натиснути **Фільтрувати**.

### Імпорт через командний рядок
```bash
python import_file.py --file reviews.csv --column text
```

## Як визначається тональність

Використовується бібліотека **VADER** — оптимізована для коротких
англомовних текстів та відгуків.

| Значення compound | Тональність |
|---|---|
| >= 0.05 | позитивна |
| <= -0.05 | негативна |
| між -0.05 і 0.05 | нейтральна |

## REST API

Повна документація доступна у Swagger: `/docs`

| Метод | URL | Опис |
|---|---|---|
| `POST` | `/feedbacks/` | Додати відгук (JSON) |
| `GET` | `/feedbacks/` | Список відгуків |
| `GET` | `/feedbacks/{id}` | Один відгук |
| `DELETE` | `/feedbacks/{id}` | Видалити відгук |
| `POST` | `/feedbacks/import/columns` | Отримати колонки CSV |
| `POST` | `/feedbacks/import` | Імпортувати з CSV |
| `GET` | `/stats/` | Статистика тональності |
